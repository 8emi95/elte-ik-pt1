Angster Erzsébet: Objektumorientált tervezés és programozás Java 
1. és 2. kötet, Közös mellékletébõl

A jar fájl hordozható, futtatható egy másik gépen.
A futtatás feltétele, hogy a gépen telepítve legyen a JAVA futtatókörnyezet, vagyis a JRE 1.4.

Két lehetõség van a futtatásra:
  1. A jar fájlhoz társítsa a bin könyvtár javaw.exe fájltját! Kattintson kettõt a jar fájlon!
  2. Állítsa be az op. rendszer JAVA_HOME környezeti változóját a JDK könyvtárra!
     Kattintson kettõt a runJar.bat fájlon!

Önálló applet fordítása és futtatása: javaapplet.jpx (javaprog mappa)
- A fordítás eredménye (class fájl) a javaprog_classes mappában keletkezik. 
- Futtatáshoz meg kell adni a html fájlt a Run konfigurációban (Project properties/Run/Edit/Html file)

Az appletek futtathatók az index.html fájl segítségével, valamint a http://4kor.hu oldalról is.

Az appletviewer segítségével megnézhetõ az applet HTML böngészõ nélkül is.

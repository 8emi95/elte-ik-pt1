Adatok

-       név: String
Aktuális tanuló neve.

-       minNevek, maxNevek: konténer(String)
 Legrosszabb és legjobb átlagú tanulók neve.

-       átlag, minÁtlag, maxÁtlag: number
Aktuális tanuló átlaga. Legrosszabb és legjobb tanulmányi átlag.

Pszeudokód

tanulmány()

    minÁtlag = + végtelen

    maxÁtlag = – végtelen

    in: név

    while (név != "*")

        in: átlag

        if átlag < minÁtlag

            minÁtlag = átlag

            minNevek törlése

            név felvétele minNevek-be

        else if átlag == minÁtlag

            név felvétele minNevek-be         
        end if

        if átlag > maxÁtlag

            maxÁtlag = átlag

            maxNevek törlése

            név felvétele maxNevek-be

        else if átlag == maxÁtlag

            név felvétele maxNevek-be         
        end if

        in: név

    end while  

    if minÁtlag == + végtelen

        out: "Nincs tanuló!"

    else

        out: minÁtlag

        minNevek listázása

        out: maxÁtlag

        maxNevek listázása

    end if

end tanulmány

Adatok

-       aktTanuló, minTanuló, maxTanuló: Tanuló(név: String, átlag: number)
Aktuális tanuló, legrosszabb és legjobb átlagú tanuló.
Egy tanulóról tároljuk a nevét és a tanulmányi átlagát.

Tevékenységdiagram

Pszeudokód

tanulmány()

    minTanuló.átlag = + végtelen

    maxTanuló.átlag = – végtelen

    in: aktTanuló.név

    while (aktTanuló.név != "*")

        in: aktTanuló.átlag

        if aktTanuló.átlag < minTanuló.átlag

            minTanuló = aktTanuló

        end if

        if aktTanuló.átlag > maxTanuló.átlag

            maxTanuló = aktTanuló

        end if

        in: aktTanuló.név

    end while  

    if minTanuló.átlag == + végtelen

        out: "Nincs tanuló!"

    else

        out: minTanuló.név, maxTanuló.név

    end if

end tanulmány
